Feature List (CMRC Train Route Builder)

•	Interactive Hex Map Route Builder
  Click hexes in order to build a train route visually.

•	Enforced Adjacent Travel Rules
  The next selected hex must touch the previous one (side-neighbor validation).

•	Route Rewind Editing
  Clicking an already-selected hex removes that hex and everything after it, letting you quickly re-path.

•	Automatic Route Display
  Shows a clean numbered route line like:
  1. Region → 2. Region → 3. Region

•	Start + End Custom Naming
  Optional text fields for First location and Last location (ex: “Depot”, “Seaport”, etc.).

•	Global + Regional Message Output
  Includes a Global Logi Message plus region-by-region travel announcements.

•	Copy-to-Clipboard Buttons
  Each message has a Copy button for quick posting.

•	Copied Highlight Feedback
  Copied message cards visually highlight so you don’t lose your place.

•	Train Flavor Text System
  Adds optional flavor like “Choo Choo!”, “Mind the gap!”, etc.

•	Flavor Modes
  o	Disabled
  o	Random
  o	Manual selection from preset list

•	Return Trip Button
  Instantly reverses the route and swaps start/end inputs.

•	Reset Button
  Clears the full route in one click.

•	Selection Counter
  Shows how many hexes are currently selected.

•	Light/Dark Mode Toggle
  Built-in theme switching with saved preference via localStorage.

•	Persistent Settings
  Theme + flavor settings persist between sessions (localStorage support).
